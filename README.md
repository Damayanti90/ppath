This github repository is meant for predicting protein-pathway associations involving dark kinases. For computational convenience, we have marked all proteins and pathways with ids starting with 'p' like 'p1', 'p234', etc. Entries from 'p1' to 'p18054' denote proteins and the rest denote pathways.

Instructions for execution

1. Download all files and place them in one folder

2. Execute the two .py files in any order.
