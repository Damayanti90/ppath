This github repository is meant for predicting protein-pathway associations involving dark kinases. 

Instructions for execution

1. Download all files and place them in one folder

2. consolidated_esm2_embeddings.txt is zipped and split into 9 parts. Kindly merge and unzip.

3. set up the environment via requirements.txt

4. Execute the following .py files.

      a. a_esm2_reduced.py

      b. b_sbert_reduced.py

      c. c_merged_embeddings.py

      d. e_ID_formation.py

      e. f_HGT.py

      f. g_XGBoost.py

      g. h_LightGBM.py
